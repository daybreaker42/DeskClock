# DeskClock 프로젝트

## 탁상시계 DIY
- 현재 시각, 현재 날씨를 보여주는 탁상 시계

## 사용 모듈
- Node MCU esp 8266
- LCD 1602 I2C display
- 아두이노 rtc 모듈
- 가속도 센서 MPU-6050
- 온습도 센서 DHT22
- 가변저항 WH148 3P
